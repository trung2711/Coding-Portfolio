Pop-up Image Slide Show and Background Selection

Welcome to my tutorial page! This tutorial shows you how to create a pop-up slide show and how to set a slide show image to be the background image of your page.

Creating the basic framework

The basic framework consist of a full-screen div element, which contains a bar at the top and a container at the middle of the screen.

Adding images

Now we want to add in the images. Using an input function to get user's path to the image, we record the path to an array and use it to create the slide show.

See the Pen Add in Framework by Trung (@trung2711) on CodePen.

Adding buttons

Next, it's time to add in the basic buttons (previous image, next image, close button. Notice that we only add in the buttons at first and have not created any functions for them. We also add some styling to the demo image so that it begins to look nicer.

See the Pen Add in Framework by Trung (@trung2711) on CodePen.

Making the framework fully functional

Now we need to take care of the functionalities. Since this is a pop-up slide show, it should be hidden when created and should only be shown when triggered (clicking on a button). The previous & next button should change the displayed image. Clicking on an image on the bar should also do the trick. We do this by setting buttons' onclick attribute (setAttribute("onclick","etc")) to trigger functions that change the display and .active option for the images in the css file.

See the Pen Add Functionalities by Trung (@trung2711) on CodePen.

To change the background

We let the user put in an id for a div of which they want to change the background. We can manipulate the div's class name and its styling in JavaScript to change its background image. The user can choose to not change the background of any div elements, and the slide show will not include the "Set as Background" button.

See the Pen Add Functionalities 2 by Trung (@trung2711) on CodePen.

For Consumer

Developer can download the framework here. It contains the JavaScript file, the CSS file, the demo.html file, and the example images used in the demo.

To use the framework, first include the .js and .css file inside the appropriate tags. Then, call the addImg() function and pass in the path to all the images (you can also call addImgArr() function and pass in a whole array of image paths instead of adding them in one by one).

Finally, call the createModal() function and pass in the following parameters (in order): the id of the DOM element used to display the modal; the id of the div element of which you want to change the background; and the height of the demo bar. The last 2 parameters are optional: the set-background button will not be created and the height will be set to a default height if you set both parameters to null.

To remove an image from the slide show, the user can call removeImg(PATH_OF_IMG_TO_REMOVE). The user can also clear the entire image array using clearImg() function. After modifying the array, call createModal() again to apply the changes.

To change the size of any other elements inside the modal, go to the style.css file and play with the styling. Do this at your own risk (could mess up the modal).
