Homework 5: Log on

In this Homework, I learned how to add log on functionality and allow users to view their profile when they are logged on. I learned how to get user input by using HTML <input> tags, how to send input to a Web API using URL parameters, how a Web API extracts the input from the URL parameters using request.getParameter("parameterName"), and how java code (safely) injects user input into a SQL statement by using the setString method of a java.sql.PreparedStatement object.

The homework was overall challenging, because sample codes were not provided as much as in the previous homework. However, things got easier after I successfully displayed user info after the user logged in (which means I have successfully retrieved the data from logonAPI.jsp). Getting the API to work was the most important task in this homework. The next problem was to process and display what the API gives me after I put in the parameters.
